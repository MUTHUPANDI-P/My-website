Medi-Bot Disease Prediction Chatbot

Project Overview

This project involves developing a chatbot that can predict diseases based on user symptoms. The chatbot interacts with users, collects their symptoms, and provides possible disease predictions using a machine learning model.

Key Features

The project involves the following key steps:

  • Designing and developing the chatbot interface
  • Building a machine learning model for disease prediction
  • Training the model on medical datasets
  • Integration of the model with the chatbot for real-time predictions
  • Evaluation and refinement of the chatbot's prediction accuracy

Technologies Used

The project utilizes the following technologies:

  • Python
  • Flask for the web framework
  • Natural Language Processing (NLP) for symptom analysis
  • Scikit-learn for machine learning
